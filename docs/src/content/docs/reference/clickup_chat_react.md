Add a reaction to a Chat message

### Synopsis

Add an emoji reaction to a message in a ClickUp Chat channel.

```
clickup chat react <message-id> <emoji> [flags]
```

### Examples

```
  # React with thumbs up
  clickup chat react msg123 thumbsup

  # React with a custom emoji
  clickup chat react msg123 rocket
```
