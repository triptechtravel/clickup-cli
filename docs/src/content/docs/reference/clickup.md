ClickUp CLI - manage tasks from the command line

### Synopsis

Work with ClickUp tasks, comments, and sprints from your terminal.

Integrates with git to auto-detect tasks from branch names.
Links GitHub PRs, branches, and commits to ClickUp tasks.
